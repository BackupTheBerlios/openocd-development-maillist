Hi,

here is a patch against OpenOCD SVN revision 184
which adds support for the Segger J-Link JTAG interface.

The necessary information about the protocol has been gathered
from logs of the USB communication.  It is rather straightforward.

I have tested it under Linux with the Atmel SAM-ICE OEM version
of the J-Link updated to firmware version Jun 28 2007 10:45:08 ARM Rev.5
and an AT91SAM7S-EK evaluation board with an AT91SAM7S256 controller.
As far as I can see it works as it should
(in particular flashing, and debugging with GDB / Zylin Eclipse plugin).
I have not yet benchmarked it w.r.t. speed.

Compared to the earlier version I sent to a few people this version
corrects the reset logic.  Now SRST works, provided it is enabled
in the AT91SAM7S controller (mww 0xfffffd08 0xa5000001).


Have fun
