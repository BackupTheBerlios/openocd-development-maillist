> I notice there is DSP56300 support - I don't know if it is complete, and

Its not complete. In the last time i have made some changes but not published yet.


The 563XX is a 24Bit and the 5680XX is a 16Bit DSP. Both have the OnCE debug interface. If you want 
to start you can use the dsp563xx_once code to access the dsp. The allowed instructions are listed 
in the AN1935 but they differ a little bit from the 563XX.
