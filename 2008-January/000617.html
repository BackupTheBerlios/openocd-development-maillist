Some notes with regard to XScale debugging:

- I've only tested PXA25x myself, and one particular PXA270 target (but that 
one only for a few hours)
- IXP42x is reported to work
- Other XScale implementations might behave differently - especially the 
PXA27x caused me some headache
- nTRST and nSRST is a must - it /could/ be possible to work without nTRST 
(having nTRST deasserted all the time), but I'm not sure if the 5x TMS 
sequence is enough for the XScale's debug logic
- mixed endiannes (host != target) has been a constant source of trouble, but 
most of these issues should be fixed by now
