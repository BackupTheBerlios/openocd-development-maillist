Trying to summarize a consensus here:

- probe() is left as is.
- add auto_probe() that is invoked get_flash_bank_by_xxx(). The driver
must do the probe if necessary and auto_probe() can fail.
auto_probe() can be invoked at any time.
- caching is implemented by each driver
- no new commands, no commands removed.
- Some drivers/targets may require an explicit probe. auto_probe()
will then fail if the explicit probe was not invoked first.
