I'm refering to all the posts to this topic, so I'm replying to the first one 
here:

- I agree with all of you that such a feature would be nice
- Like Øyvind I'd prefer to see this implemented as commandline options, not 
environment variables
- Please (try to) make it work with
	- Linux
	- Windows (Cygwin and MinGW)
	- OSX (at least Intel)
	- FreeBSD
- There could be all kinds of hidden problems when allowing all commands to 
execute during configuration (Øyvind mentioned that change in some post). The 
daemon startup pretty much relies on CONFIG commands executing before EXEC 
commands.

Best regards,

Dominic
