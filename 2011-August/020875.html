vsllink.c defines the 'vsllink_debug_buffer' function when verbose USB
comms or verbose JTAG comms are enabled, but each use is guarded by
#ifdef _DEBUG_JTAG_IO. So the function is unused if only verbose USB
comms are enabled, and this is an error with -Werror. I fixed the
problem as described in the following patch.
