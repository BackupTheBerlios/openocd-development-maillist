On Wed, Jul 15, 2009 at 10:43 AM, Qiang Wang<rurality.wq at gmail.com> wrote:
> I am studying the open-ocd and the JTAG also.
> I know that this project use the FT2232 chip.

No. This project is not using FT2232 chip. Some of the JTAG debuggers
supported by OpenOCD happen to use FT2232x chip. But OpenOCD
supports other JTAG debuggers which do not use FT2232x.

> But I can not get the clear architecture about the open-ocd ,FT2232
> and the target .
> Can somebody tell me or give some document link.
>

OpenOCD is the software which drives the J-TAG debugger.

The target supported by OpenOCD is mainly the ARM based
chips with built-in JTAG circuitry.

For more information, you can refer to the documentation.
http://openocd.berlios.de/doc/html/index.html
