in cygwin you can use gcc in two modes:
1. just calling cygwin 'gcc' will link to cygwin1.dll and other cygwin posix
dll's
2. if you call the cygwin 'gcc -nocygwin' option then this becomes a native
win32
binary just the same as if you used msys/mingw to build

And from experience cygwin will be easier to setup then msys/mingw.

Michael's tutorial explains all the details of using both options:
http://forum.sparkfun.com/viewtopic.php?t=11221
