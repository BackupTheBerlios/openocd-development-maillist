> Do we have other data for other JTAG debuggers? For example, J-Link?

We found that switching from openocd r1609 to openocd r2529 using
libftdi (0.13) with either amontec JTAGkey or olimex USB-tiny the
upload speed approximately doubled. (from 4MB in 6 minutes to 4MB in 3
minutes)

The new fast speed approximately equals the performance of amontec
JTAGKey on Windows with the amontec-supplied libftd2xx and r717-ish
openocd.

I don't have the actual numbers to hand but can post them when I get
back if this is interesting.

I don't know if this speed-up is to be expected? It is certainly very
welcome.
