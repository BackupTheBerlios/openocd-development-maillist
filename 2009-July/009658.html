Maybe you are missing the libtool for MinGW/MSys.

It is possible to build OpenOCD with MinGW/Msys without Cygwin.
You need to install some extra stuff other than the core MinGW/Msys.
https://lists.berlios.de/pipermail/openocd-development/2009-July/009657.html

I have just built the ftd2xx version without any problem.

$ ./configure --disable-shared --enable-jlink --enable-ft2232_ftd2xx --with-ftd
2xx-win32-zipdir=/home/mcuee/ftd2xx
