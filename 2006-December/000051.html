All, I recently acquired the AT91SAM7SE-EK with an AT91SAM7SE512 processor.
Not suprisingly OpenOCD didn't support this new processor in terms of flash
support. I made the changes to support the flash in all three of the
AT91SAM7SE processors. I was able to test with the AT91SAM7SE512 but the
other two processors are untested. I've attached the unified diff for
src/flash/at91sam7.c to this message. Hopefully these changes can be merged
into the trunk so that others won't have to modify the code in the manner
that I did.
Cheers.
