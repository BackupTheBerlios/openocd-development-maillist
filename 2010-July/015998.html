BTW, I also tried your 32bit development package with the
libftdi-0.18 32bit MinGW dll I uploaded, it seems to running fine.

Take note that you can run 32bit OpenOCD under 64bit
Windows. The only thing is that you need to use 64bit
drivers. You can not use 32bit drivers under 64bit Windows.

Your 64bit packages need to use the 64bit of libftdi dll so I can not
test with libftdi-0.18 yet.
