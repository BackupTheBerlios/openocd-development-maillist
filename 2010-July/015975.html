There might be incompatibilities between the libftdi version and
the libusb-win32 version. I need to carry out some investigation
myself to see if the 1.1.14.3 version is compatible with the
libftdi.dll from Freddie (libftdi-0.17). If not, probably I will
post a new libftdi.dll myself (probably libftdi-0.18).

In the mean time, please try to use the libusb-win32 device driver
version packaged in Freddie's version (0.1.12.2). You should not
need to copy the dll to the C:\Windows\system32 by yourself.
Something is wrong in your driver installation procedures.

Please follow Freddie's instruction. If you are using 32bit
XP, you should find libusb0.sys under C:\Windows\System32\drivers
and libusb0.dll under C:\Windows\System32. And the version
should be 0.1.12.2 for both files.
