This looks like only a 74xx125 is needed to convert any existing JTAG 
dongle to support SWD mode. Most of the switching can be omitted if you 
need only fixed JTAG to SWD conversion - if you need standard JTAG, you 
can just remove the adapter and use the existing FT2232 dongle as-is.
