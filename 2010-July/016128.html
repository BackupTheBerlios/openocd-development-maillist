Attachment is the patch for new Versaloon driver, but it's ONLY a
preview version. The new driver uses generic USB_TO_XXX protocol
designed for Versaloon. And the USB_TO_XXX protocol driver code
is exactly the same as in vsprog. So I can simply mainatain one
driver framework, and new driver of other transports can be simply
added.

Releated code in Versaloon firmware can be accessed via
http://code.google.com/p/vsprog/source/browse/trunk/dongle/firmware.
For SWD code:
http://code.google.com/p/vsprog/source/browse/trunk/dongle/firmware/Interfaces/SWD
.
