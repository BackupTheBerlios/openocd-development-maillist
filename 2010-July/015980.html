> compatible with some USB device).

Ok, I just tried the same thing under Windows 7 x64,
it seems to work as well. Again, I am using the default
FTDI driver provided by the vendor (TI/Luminary) and the
libusb-win32 filter driver. I am enabling the test signing
here in order to use libusb0.sys.

We will release a signed libusb0.sys which can be used
in 64bit Windows Vista and Windows 7 pretty soon.
