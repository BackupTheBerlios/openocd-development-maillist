Hello?

My name is unsik Kim at mGine co. Thanks for great tool.

I made a patch for mFlash. (mFlash is a fusion memory that has SRAM
interface and uses a ATA protocol. Simillar product is sandisk mDoc
but protocol is different.)

Currently some navigation, settop box, cell. phone company uses
mFlash. (not many T.T; )

Using patch, user can probe, write and dump mFlash. mFlash is a active
device, so mflash support is somewhat host SOC dependent (cause of
reset, wright protect, deep power down).

Currently patch is somewhat poor, but I improve constantly.

Patch file is made from svn rev. 1062.

In openocd root dir, patch -p0 < mflash.patch

Please don't hesitate for your opinion.. or whatever.

p.s. patch file also include target_write_u8 's little bug fix.
