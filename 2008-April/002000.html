Are there any plans afoot to get FT2232 'layout' information out of
source code and into configuration files? If not I'll try and russle
something up for submission but it might take some time.


In the simple case (no inputs), all that's needed is:
 
1) GP Output Mask
2) GP Output Value
3) nTRST Mask
4) nSRST Mask
5) Connected LED Mask
6) Running LED Mask


For example, we have an in-house FT2232 debug board for our PXA2xx
platforms; the 'Generic FT2232 Driver' layout would look something like
this:

Output Mask        = 0x0ffb;
Output Value       = 0x0708;
nTRST Mask         = 0x0200;
nSRST Mask         = 0x0100;
Connected LED Mask = 0x0800;
Running LED Mask   = 0x0300;
