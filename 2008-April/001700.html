I wrote up this patch and it compiles, but I haven't tested it. Testing,
as always, greatly appreciated! :-)

Thanks to Pavel for coming up with the concept!

- add "init" command. Executing this from within a startup script will
switch from config to normal execution. Issuing "init" a second time
is a no-op.
- retired "daemon_startup".

Example:

1. openocd -f target/foo.cfg -c init -c "reset run"

=> run target

2. openocd -f target/foo.cfg -c init -c reset -c "flash write_image
c:\xxx\foo.elf"

=> write a file to flash
