I'd like to enable DCC memory downloads by default.

Perhaps this option should be retired entirely and always
be enabled?

This simplifies documentation in all that is required for decent
performance is working memory, which is in line with anyones
expectations after using other ARM JTAG debuggers.

Without working memory enabled, DCC memory downloads
are not used.

Any objections?
