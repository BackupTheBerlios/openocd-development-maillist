This patch allows launching OpenOCD w/telnet+gdb server w/the
target powered down.


This can be useful when dealing with JTAG controllers that need
a bit of prodding to get going(e.g. LPC2148) and also
remote usage of OpenOCD.


Test:

1. power down target
2. launch openocd
3. launch gdb session
4. power up target
5. (gdb) monitor reset
