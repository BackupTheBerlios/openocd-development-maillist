Hi,

Decided it was finally time to update the driver - its been on the list for
a while.
The flash banks have been split up, similar to the str9 driver, this makes
it behave correctly as the flash driver logic expects consecutive banks.

Joan, feel free to test and let me know how you get on.
The only change is both banks will need declaring, for example.

flash bank str7x 0x40000000 0x00040000 0 0 0 STR71x
flash bank str7x 0x4000C000 0x00004000 0 0 0 STR71x

to erase the complete device
flash erase 0 0 7		/* erase bank0 */
flash erase 1 0 1		/* erase bank1 */

I am not committing yet, as all str7x hardware is at the office, will fully
test next week.

Cheers
Spen
