On Tue, Sep 14, 2010 at 12:58 AM, Freddie Chopin <freddie_chopin at op.pl> wrote:
>> However, can you please point me to the drivers (inf files) if they
>> are already present in OpenOCD tree, so that I do not have to use
>> http://sourceforge.net/apps/trac/libusb-win32/wiki tools at all.
>
> ...\OpenOCD\0.4.0\drivers\libusb-win32_ft2232_driver-100223.zip

This is not that good since it refers to an older version 0.1.12.2 which
may have some problems, especially if the user has the filter service,
then it can cause BSODs. And this one is not digital signed, so it
will not work under 64bit Windows 7 and Vista out of the box.

> ...\openocd-0.5.0-dev-100827221639\drivers\libusb-win32_ft2232_driver-100831.zip

This is much better as it is based on 1.2.1.0. It is digital signed
and should work under  64bit Windows 7 and Vista right out of the box.

Still for the end users, it may be easier to use libusb-win32
inf-wizard or libwdi/zadig for the driver installation.

libusb-win32:
http://sourceforge.net/projects/libusb-win32/files/
(download the libusb-win32-bin-1.2.1.0.zip )

libwdi/zadig
http://www.libusb.org/wiki/windows_backend
