But it isn't a harmless addition that I can safely ignore.  Now when a  
new file or folder gets added, I need to modify both Makefile.am as  
well as the CMake files.  It add extra time to _my_ development for no  
immediate benefit to me.  Further, anytime the 2 build systems get out
