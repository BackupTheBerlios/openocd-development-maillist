I'll try to implement a more generic code.

Attachment is the current ugly simple code.
avrt.c and avrt.h is the avr target support(in target directory).
avrf.c and avrf.h is the avr flash support(in flash directory).

For a generic support for AVR or even AVRXmega:
avr.c and avr.h is code for avr core in target directory.
avrf.c and avrf.h is code for avr flash driver in flash directory.
avr_common.c and avr_common.h is data for every avr chip in target directory.
avrx_common.c and avrx_common.h is data for every avrxmega chip in target directory.
avrx.c and avrx.h is code for avrxmega core in target directory.
avrxf.c and avrxf.h is code for avrxmega flash driver in flash directory.

I haven't developped or maintained a target chip support of OpenOCD.
Mail me if any suggestions about adding support to avr or avrxmega.
