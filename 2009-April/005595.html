- restore indentation that was flattened by a previous commit

- disable IRQs while stepping

- add "arm11 step_irq_enable true/false" command that enables IRQs
during stepping

- add "arm11 no_increment true/false" command to allow reading/writing
data blocks from/to a fixed address (for example a port)

- where TAP_INVALID is used to choose the default end state of a
function use ARM11_TAP_DEFAULT instead to be more descriptive


I have attached two files:
- arm11-vs-r1511.gz   is the actual patch
- arm11-vs-r1511-readable.gz  are just the non-white-space changes for
better readability
