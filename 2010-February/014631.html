On Sunday 07 February 2010, Julien Iguchi-Cartigny wrote:
> - at the present time we are using a jtagkey-tiny, low price usb/jtag
> but with a small problem: it seems this dongle require root rights on
> the host. Does anyone know if it's possible to use it without root
> rights (like giving extended rights to use some usb devices) ?

If you're using Linux, I suspect you just forgot to install
the UDEV rules which automatically set up access for you.

 
> - does anyone has some recommendations for a cheap usb/jtag which can
> run without privileged user on the host ?

Privilege is an OS-specific issue; but, see above.  I don't
have privilege issues on Linux, with the UDEV rules set up.


> - in the educational context, what is your recommendations for a cheap
> ARM cards which work without a glitch (i.e. we are looking for
> stability, even if the card do not have some extra features) ?

You won't find any vendors crowing about their glitchey boards.  :)

Look at Stellaris eval boards too.  There's variety in them; you
could get some of each, supporting different kinds of projects
and peripherals, while saving the cost of discrete JTAG adapters.
The very lowest cost boards (LM3S811 EK) are about $US 50, which
is less than many discrete JTAG adapters...

Cortex-M3 (from whichever vendor) is IMO a good choice.
