On Fri, Feb 19, 2010 at 5:00 PM, Spencer Oliver <spen at spen-soft.co.uk> wrote:

> The latest jlink version i have here is v8.

That is quite new. Here at work I have seen a few V3 (IAR), many V6
(Segger, IAR and Atmel Sam-ICE), V7 (Analog Device mIDAS-Link)
and one or two V8 units. We also got one J-Link Pro with the Ethernet
interface. But my colleague says that the J-Link Pro is not really
that more useful than the normal version of J-Link.
http://www.segger.com/cms/j-link-arm-pro.html

I've once seen a web page comparing different version of J-Link
but now I could not find it. V1-V4 seems to be old which
uses Renesas MCU. V5/V6/V7/V8 uses Atmel ARM MCUs.
The Pro also uses Atmel ARM MCUs. There are some hardware
differences between different revisions. Firmware features
are also a bit different. Some OEM versions are locked to
the specific vendors (like Atmel or ADI device) when using
IAR.
