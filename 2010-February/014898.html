For interface, it just tries all the interface definitions in /usr/local/share/openocd/scripts/interface.
Whichever one is detected last is then loaded with an empty scan chain, and it should report all the detected TAPs on the chain. Note, that's just the test code which tests a wrapper class. Look at the bottom in the "if __name__ == '__main__':" section.
