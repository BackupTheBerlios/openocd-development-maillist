On Thursday 11 February 2010, Igor Skochinsky wrote:
> On thing I like about LM3S811 kit is that it has a couple of
> peripherals on-board: LCD, a button, a LED and a potentiometer
> connected to ADC. Thus you can do some interfacing even without extra
> circuitry.

Which is VERY nice for beginners.  The other Stellaris eval
boards are similar ... but all have more peripherals (and
often more memory too).  Bigger LCD, Ethernet, USB (host,
peripheral, OTG), rocker switch, and so on.


> The chip itself is somewhat less capable that NXP's; for 
> example, no USB support.

Depends which LPCXpresso board you look at.  There are two;
the LPC1114 (Cortex-M0) board has no USB, and no current
OpenOCD support.  The LPC1343 (Cortex-M3) board has USB,
and one expects OpenOCD support would require minimal work
(maybe just new config files).

Though as you note, neither one yet works without CodeRed
tools on MS-Windows ... evidently virtualized Windows can
be used from Linux (what a PITA).  And then there's the
obstacle of a proprietary JTAG adapter.
