changed the system config files but i dislike doing that.

My suggestion was to have possible aliases. So then the config would be 
something like:

ft2232_device_desc "Olimex OpenOCD JTAG TINY"
ft2232_device_desc_alias "Olimex OpenOCD JTAG TINY A"

Indicating that any of these description names refer to the same device 
really.
