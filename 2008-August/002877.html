I have not yet used dataflash, but in case you can get RAM initialized from OpenOCD, one solution might be to set up the RAM controller from openocd, and then download & run some code (with embedded target image) that copies the image data to flash. At least that's how I usually start on IXP42x targets: download a RAM version of my bootloader, and use that to program any flash - even those that are not (yet) supported by OpenOCD.
