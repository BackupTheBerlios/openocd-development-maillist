Hi,

I have some trouble getting OpenOCD working. I have tried different
combinations of targets and debug interfaces:
 - STR9-comStick
 - IXP425 board
 - Olimex ARM-USB-OCD
 - selfmade parport wiggler, built according to the schematics in thesis.pdf

After some struggling, I got the combination of Olimex USB + IXP425 running
(which was my main goal) - I can talk to the target, display memory, etc.

When I try the same using the parport interface, initialization of OpenOCD
fails quite early:

Error:   jtag.c:1236 jtag_examine_chain(): JTAG communication failure, check connection, JTAG interface, target power etc.
