I don't know anything about the iMX27, but I've got an
iMX31 in my office where the JTAG chain now enumerates
properly.

The iMX31 JTAG has shown evidence of being finicky w.r.t. the analogue
part/precise implementation of the JTAG protocol. Same OpenOCD code will
work on one JTAG interface and fail on another.

If there are two devices in the chain, then you'll need two
jtag_device statements
even if only one is used.
