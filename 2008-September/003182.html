Wow!

This is a *completely* new driver.... There are *LOTS* of changes here...

This is a problem from a regression testing point of view....

Regression testing won't happen in the community until this is committed, yet
we don't want to risk breaking those that dare run their development tools
out of openocd trunk as we need their testing feedback.

What I suggest is that instead of modifying  the old driver, you duplicate the
source files and create a "new_at91sam7.*" driver.

This allows the old and new drivers to be compared with relative ease.

At some point regression testing will be done and we can retire the old
driver.

How does that sound?

If you agree, please make the chaneg and repost the files to me and I'll
commit the new at91sam7 driver.
