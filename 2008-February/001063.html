This is another step towards a target library.

- added -c option that will execute an openocd command
- added at91eb40a target library example.

Example:

openocd -f target/at91eb40a.cfg -c "jtag_speed 0"


This will choose the at91eb40a.cfg config script from the
library and tweak jtag_speed option.

Known problems:

- the target library should search relative to openocd executable
  so the target libraries are found. This fix can also be used to
  solve the  problem wtih debug_handler.bin not being found
  under Windows
- rename the repurposed config_file_names construct to
  startup_commands_xxx or somesuch. Rename not
  done this time around to reduce patch noise.
- interface/*.cfg examples are missing. Patches gladly
  accepted!
