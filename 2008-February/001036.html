Hello Michael,

you could use a statemove from PD to PD between the two dr_scans. Statemoves 
from a pause state back to the same pause state go through E2D, UD, SDS, CD, 
E1D and end in PD again.

This is currently implied behaviour, but all currently existing JTAG interface 
should support it. This is something that should be documented as part of the 
JTAG layer, if there are no objections to defining such constraints to what a 
statemove does.

Regards,

Dominic
