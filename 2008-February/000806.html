Hi,

Attached is a small patch that should fix a problem flashing the smaller
lpc, eg. LPC2101/2102
The issue was because the current flash driver will try and allocate a
4096+172 byte buffer, which on these devices is not possible as they only
have 2048/4096 total ram.

I cannot test as i do not have access to silicon, but the changes are so
small it should not cause a problem.

If anyone could test it would be good, before i commit.
I have forwarded the patch to the person who reported the problem, so he
should be able to test aswell.

Cheers
