I'd like to see working_area deal with RAM moving after the MMU has
been enabled.

How about adding an optional argument for a physical address?

If the MMU is enabled the virtual address is used, otherwise the
phsyical address is used. Default physical address is the same
as the virtual address.

Note that the virtual address does not need to point to the physical
memory area used.

Any better ideas? Objections?
