I always build with -Wall on, just by overriding on the command line.

There have been debates about this on other projects including gcc/gdb - the
outcome is let the developer choose what flags he wants.

Imposing default flags can effect the flags you are actually interested in,
very unlikely in the case of Wall however.

The user who just wants to build the code from source will just use the
default of -O2 -g.
