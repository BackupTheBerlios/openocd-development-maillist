Don't have one, but ISTR it's more or less
an OMAP3 flavor aimed at indistrial apps.
SO treat it as an OMAP3 at first.

(Distinct from a DaVinci with Cortex-A8...)

I don't think current generation TI chips
use anything except ICEpick-C ... not much
public doc on that, try using the current
support for it.  9 (EMU0/EMU1 setting info
is with other pinning info in the data
manual (for board designers), not the TRM (
which is for programmers)  Usually there's
a jumper block or DIP switch for those signals.
