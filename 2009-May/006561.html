The OpenOCD project presently uses the mailing list and Subversion.
Personally, I am not eager to change that status quo right now, but
perhaps we can revisit this topic in the future.

I did not plan to incorporate The List into the Doxygen manual; however,
I sold myself on the ability to link into the code documentation
directly from the todo items (without having to do anything special).
In other words, referencing 'function()' will automatically create a
link to the doxygen documentation page for that function.

In saying this, I realize that my first version failed to do this right.
I've started to make a patch, but I am not going to rush to commit it.
Right now, others should have a chance to offer some updates.
