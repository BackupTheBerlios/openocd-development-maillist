Based on the lpc3180 driver, but released as a separate driver for two reasons:
 1) I don't have an lpc3180 to test it against, so it might not be compatible.
 2) It's using a different OOB layout than lpc3180

Also borrowing code from the NXP CDL library, written by Kevin Wells.

Tested on lpc3250 (Hitex LPC3250-Stick).  OOB layout is compatible
with LPCLinux.
