After some experimenting with building the right cable and generating an appropriate SVF file, I was able to program a Lattice XP2 using OpenOCD and a Signalyzer-H2. Since I also have a jlink, I figured why not give that a shot.  It seems that the jlink does not have the JTAG_STABLECLOCKS jtag command implemented which is required for this SVF file.
Poking around jlink.c, seems to indicate that JTAG_STABLECLOCKS could be implemented using jlink_tap_append_step() if you knew what TDI should be during the clocking.  Any clues on how to figure out what TDI is supposed to be?
