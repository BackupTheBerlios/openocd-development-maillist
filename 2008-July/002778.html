Committed.

Minimum address and maximum length argument to load_image. Used
in lieu of reset init script when executable w/reset init sequence is available.

Typical usage would be to take a working flash image, e.g. compiled with
eCos or some other embedded os/library, then load the beginning of
that image into sram available upon reset halt(before remap takes place).

That code will probably be position independent.

Step through enough instructions or set a breakpoint and voila! you should
have your memory map configured.

Probably not as useful for flash + dram scenarios though.

load_image <file> <address> ['bin'|'ihex'|'elf'|'s19'] [min_address]
[max_length]
