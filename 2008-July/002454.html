The problem with the luminary targets is that they are all very similar.
I have added a libftdi config to openocd for the luminary eval boards.

To work with the LM3S8962 you would issue the following:

openocd -f interface/luminary-libftdi.cfg -f target/lm3s6965.cfg
infact you could also use
openocd -f interface/luminary-libftdi.cfg -f target/lm3s811.cfg

only difference is the size of working area.

we could add a generic luminary config - target/luminary.cfg for example
but each target would still need the correct working area set.
