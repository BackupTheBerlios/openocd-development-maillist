Oyvind,

Earlier I said that we needed some support for the "array" feature in 
Jim TCL.
Actually - we do not, we have enough there now - by other means!

It can be accomplished as follows:

 >  set foo(me)  Duane
 >  set foo(you) Oyvind
 >  set foo(mouse) Micky
 >  set foo(duck) Donald

If one does this:

 >  set foo

The result is:

    me Duane you Oyvind mouse Micky duck Donald

Thus, to get the names of the associative array is easy:

     foreach  { name value }   [set foo]   {
                puts "Name: $name, Value: $value"
     }

Using that simple example - one can construct a tcl function to 
"extract" the names of an array.

-Duane.
