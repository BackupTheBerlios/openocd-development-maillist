Hi all,

here is a patch for handling F1 - F12 keys, each running a customizable 
script file.

It also implements an escape sequence parser that simplify a lot adding new 
escape sequences.

I'm a little confused about different telnet key mapping between Win and 
Linux and I tryed to support both.

My F11 script is used to single step, disassemble some lines @PC, print 
registers.
To do that I also modified a little the file armv4_5.c to be able to 
disassemble at current program counter, and I also corrected it to be aware 
of the current processor state (ARM/Thumb).

I also attached my script dir to handle F1 - F12 keys.

Please, feel free to modify/rewrite/reformat the code.

Regards,
          Dario
