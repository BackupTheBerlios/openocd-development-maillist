Hi

As I mentioned in earlier mail, I made a speed optimization for flash 
erase on STR9x.
But I guess similar improvments can be made in several other flash drivers 
aswell.

This is the result when erasing my board:

Before

flash 'str9x' found at 0x00000000
cleared protection for sectors 0 through 7 on flash bank 0
erased sectors 0 through 7 on flash bank 0 in 9.189396s

Patched (about 20% faster erase)

flash 'str9x' found at 0x00000000
cleared protection for sectors 0 through 7 on flash bank 0
erased sectors 0 through 7 on flash bank 0 in 7.180365s

The patch translates several ERASE_SECTOR command into one single 
ERASE_BANK command if whole bank should be erased.
(Please do check tab/space indentation if committing this.)



Kind Regards
Fredrik
