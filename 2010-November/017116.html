> Has anyone taken this plugin for a spin?

I'm using it, an I've even contributed some patches a year ago. Quite 
nice thing, now Eclipse can compete with commercial toolchains:
1. It has debugging support with OpenOCD + Zylin / GDB H/W debugging
2. It can automatically generate Makefiles - 
http://sourceforge.net/projects/gnuarmeclipse/
3. It has peripheral registers viewer

This plugin has nice features regarding register values, as you can 
define "sub-bit-groups" within one register (like bit 0-7 PLL 
multiplier, bit 8-10 PLL divider, etc.) which can be displayed 
separately, and you can even provide meaningful values for such 
"sub-bit-groups" (e.g. 0x0 will be displayed [an can be selected to be 
written back to register] also as TIMER_OFF, 0x1 as TIMER_UP, 0x2 - 
TIMER_DOWN, ...)

4\/3!!
