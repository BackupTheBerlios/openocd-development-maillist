Hi,
in attachment there is a first support for the STMicroelectronics
device SPEAr310 and for the ST board EVALSP310 rev. 2.0

The configuration scripts has been split in SPEAr310 specific and
SPEAr3xx family generic.
This should help adding support for the other members of the family.
My plan is to add SPEAr300 and SPEAr320.

The original boards don't provide SRST on the JTAG connector.
There is a preliminary application note (not yet available on-line)
that explains how to fix the board.
I have created 2 board files for the original and the modified board.
HW modification list is reported in the board script.

The board EVALSP310 hosts 4 non-volatile memories:
- parallel CFI NOR flash
- serial NOR flash
- NAND
- I2C EEPROM
The patch currently supports only the first one.
The support for the serial NOR is almost ready. I'm testing it.
NAND would be the next one.

Comments are welcome.

Best Regards,
Antonio Borneo
