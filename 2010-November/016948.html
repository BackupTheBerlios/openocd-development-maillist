Hello,

Over the past weeks, I've been working on and off to get BSDL-based flash access
incorporated in OpenOCD.  In its current state it isn't worth publishing, but
some of you might be interested in trying it anyway.  So I setup a branch, as
0yvind suggested to me.  It is located at:

	git://git.vanrein.org/openocd-genbsdl

The main changes are in doc/bsdl.txt (explaining the idea in more detail) and
src/target/generic_bsdl.c (implementing generic access).  The current version
has no BSDL compiler, so descriptive structures have been hard-coded into the
source file!  Integration with the rest of OpenOCD is workable, but the code
that actually runs the boundary scans is *very* much work in progress, to a
level that I haven't gotten it to work yet.

Having warned you, please feel free to experiment!  Feedback and patches are
kindly welcomed, of course :)
