normal. As soon as it reaches zero, it should terminate.

The slow speed is normal in the case where no support code on the target
is available - we need to shift several 10-100 bits for each bus cycle
to the flash, and every word takes 3 bus cycles if the flash has no
pagemode, or 1 cycle + setup if pagemode is used.
