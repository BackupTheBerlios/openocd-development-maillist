What I just posted has the M0 as a complete copy and paste of the M3 code,
so the entire thing is duplicated. However, that was more because we wanted
to test the M0 code but still be able to work with the M3. The M3 is almost
a perfect super set of the M0, so I don't think it would be hard to avoid
duplicating everything. However, there are a couple of parts where it would
be unavoidable, specificly the Thumb assembly functions. There are also are
some more sutle differences (reset handleing, registers, tracing support,
interrupts) but none of them are too dramatic.
