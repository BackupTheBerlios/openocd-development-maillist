Hi,
This patch update the driver of Versaloon.
I add a versaloon directory under src/jtag/drivers, and put all original
versaloon drivers there.

Supported usbtoxxx drivers in this patch:
usbtogpio.c -- used to control SRST and TRST lines
usbtojtagraw.c -- used to control JTAG
usbtoswd.c -- used to control SWD
usbtopwr.c -- used to set and get target voltage

Mail me if any problem.
