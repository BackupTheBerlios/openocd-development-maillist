Our intention is not for everybody to be *required* to have all these
tools that we run on the server locally. Part of the genius with Gerrit
is that the patches are not marked "verified" before all the checker
tools we can dream of have given the patch an all clear. Just amend
the commit and push to the server again for a new test build on the
server.

checkpatch is run without any options.
