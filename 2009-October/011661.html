Darn.  You gave away the secret juju.  Now *everyone* will
be able to submit patches...


> 1. First time:
> 
> sudo apt-get install git
> git clone git://openocd.git.sourceforge.net/gitroot/openocd/openocd
> cd openocd
> 
> 
> 2. To update:
> 
> git pull
> 
> 3. Make modifications, then:
> 
> git diff >myfixespostasattachmenttolist.txt
> 

Note that this "make modifications" doesn't involve committing
changes.  If you permantly modify your local repository, that
"git diff" won't show any changes.
