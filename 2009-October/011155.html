I think I'd like to remove gdb_detach command.

The current default behavior when attaching GDB is to do
nothing. The GDB init script can e.g. reset or halt the
target as need be.

Similarly, I'd like to make gdb detach do nothing. If need
be the user can do nothing, halt, rest, etc. if he wants
to, no need for a special command or option that I can see...

Comments?
