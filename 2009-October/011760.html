Software: openocd-0.3.0-rc0 built with Cygwin using the following switches:

--enable-maintainer-mode --disable-werror --disable-shared --enable-ft2232_ftd2xx --with-ftd2xx-win32-zipdir=/tmp/ftd2xx CC="gcc -mno-cygwin"



Hardware: Atmel AT91SAM9260 CU, Intel JS28F128P30T85 128MBit StrataFlash



Script: board/unknown_at91sam9260.cfg

>reset init
