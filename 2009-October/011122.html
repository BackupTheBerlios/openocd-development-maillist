We have lots of these codelets around OpenOCD:

if { [info exists CHIPNAME] } {
   set  _CHIPNAME $CHIPNAME
} else {
   set  _CHIPNAME pxa270
}

Does anyone find this the syntax below simpler or just
more obscure?

-expected-id [defaultvalue CPUTAPID 0x12345678]

defaultvalue returns either the _CPUTAPID value or 0x12345678 if
_CPUTAPID is not defined.

proc defaultvalue {name value} {

   if { [uplevel "info exists $name"} {
      return [uplevel $name]
   } else {
      return $value
   }
}
