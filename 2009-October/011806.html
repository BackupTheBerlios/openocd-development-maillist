Hello,

I've put together the spec file and a series of RPMs for submission to
Fedora once 0.3.0 is out the door.  These are built based on the rc0
tarball.  Anyone out there with Fedora who can test one of these will be
much appreciated.

http://files.dinoprojects.com/openocd/rpms/

For some of the developers, here is the configure line.  Can you let me know
if I'm missing any important interfaces/features for a general release to
Fedora?  Thanks.

%configure \
  --disable-werror \
  --enable-static \
  --disable-shared \
  --enable-parport \
  --enable-parport_ppdev \
  --enable-ft2232_libftdi \
  --enable-ep93xx \
  --enable-at91rm9200 \
  --enable-usbprog \
  --enable-presto_libftdi \
  --enable-jlink \
  --enable-vsllink \
  --enable-rlink \
  --enable-dummy \
  --enable-gw16012 \
  --enable-amtjtagaccel \
  --enable-arm-jtag-ew

// Dean Glazeski
