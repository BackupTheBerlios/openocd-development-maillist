I'm currently using ELDK ARM on a Debian host (have to install alien pkg to
deal with RPMs) and CodeSourcery g++ lite.  I like EMDebian and OpenEmbedded
too but the install of ELDK is as simple as:

Mount iso as loopback
run ./install -d /opt/eldk arm
export arch=ARM
export $CROSS_COMPILE=arm-linux-
set path to include /opt/eldk

... go start cross compiling ... in 15min tops!

ELDK is pretty complete too.

I used to do ARM7TDMI but that was 10 years ago (ENEA OSE) ... I mostly
focus on ARM 9 these days so don't know how much this helps you.
