For the record, 0.1.0 was released from the openocd-0.1.0 tag.  The  
openocd-0.1 branch is for all 0.1.x releases.  If we decide to do a  
0.1.1, the changes will be merged into the openocd-0.1 branch.  The  
openocd-0.1.0 tag will not change as it is frozen with the sources  
released for that version.

So, in general, build from the tags rather than the branches unless  
you are developing for an upcoming release.
