There is an implicit 'init' command run after all the command line  
arguments are processed.  If you do 'openocd -f interface/jtagkey.cfg - 
f target/stm32.cfg', then it will do an init just before the telnet  
server is started.  By adding '-f openocd.cfg' at the end, your script  
is run before the init.  Just add 'init' as the first line of your  
openocd.cfg and it should work.
