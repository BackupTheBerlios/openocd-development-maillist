We do install them in a common location, but you still need to tell us  
which files to load.  The standard configs are just installed in the  
standard search path.  So, '-f target/at91r40008.cfg' looks for target/ 
at91r40008.cfg in the current directory and then in the search paths.   
We split the configs into interfaces, targets, and boards to make it  
clear which is which.
