We need more target CPUs to regression test against.

We already have a respectable test-farm that we're looking
to increase in width and depth.

Thanks for contributions so far!

Please email me if you have a board that you are able to
donate. I'm sure that there are boards dusting away out
there that can get a second lease of life serving the OpenOCD
community as test targets!

It doesn't matter if it is an "official" eval board or just some
equipemnt which can server to power up a CPU w/a
JTAG connection. Some of the boards we have been
given are "broken" in their primary function, but the
CPU+JTAG works fine.

Documentation is nice, but it is also useful simply to
be able to test the actual CPU part w/JTAG without
talking to any other parts on the board.


Thanks!
