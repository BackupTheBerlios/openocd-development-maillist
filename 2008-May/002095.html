I'm open for more opinions on this topic. My current point of view is that the 
driver should go in, because it's completely free software (I assume it is - 
i.e. it doesn't link some binary-blob from the vendor?), even if it's for 
proprietary hardware.

I've been told that the driver was created without knowledge of the binary 
windows drivers, and thus it should be safe to include.

And, well, yeah, there shouldn't be support for ATI video cards (or NVIDIA, 
which is a lot worse than ATI/AMD), to increase the pressure on those 
manufacturers to release free drivers, but I doubt the OpenOCD can create 
that pressure ;)

Dominic
