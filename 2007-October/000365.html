Hi,

I have just committed the flash auto_erase patch to svn
Default behaviour is off, flash erase will need calling prior to writing
flash.
With flash auto_erase on the required flash banks will be erased
automatically when using the flash write image command only. Command has
been added to OpenFacts.

Also included are a few memory allocation bug fixes and improved hex file
reading.
