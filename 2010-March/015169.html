Hi,
This patch greatly simplify the Versaloon driver, which reduce the code size
from more than 50K to less than 28K.
And this patch also add support to IR/DR scan with unlimited size.
Meanwhile I remove the tms path maintained in Versaloon driver and use
tap_get_tms_path and tap_get_tms_path_len.
Next patch will implement USB_TO_XXX framework for Versaloon.
And use USB_TO_JTAG_RAW for JTAG, USB_TO_GPIO for SRST and TRST control,
USB_TO_DELAY for delay control.
Under this framework, USB_TO_SWD/USB_TO_SPI/USB_TO_IIC can simply be added.
I'll then add AVR_ISP support to test USB_TO_SPI.
