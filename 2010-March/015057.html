On Sunday 28 February 2010, Gene Smith wrote:
> I was never able to get the serial port on the Olimex to work with 
> linux (but never really needed it).  Maybe user error?

Check "dmesg" output to see which port it was configured to be.
Likely /dev/ttyUSB0 would suffice ... unless you have other
USB-to-serial adapters hooked up.

Then you'd need to configure minicom (or whatever) to use the
right logical TTY name.  I just save a "usb0" config so I can
pass its name to minicom as I start it up ... since I have a
very limited number of usb-to-serial adapters, and only use one
of them at a time.
