Looks like it was user error.  My working space included
non-existent ram, because I lifted the 811 file, which I wrongly
assumed was also a 4k part.

For my part,
# 2k working area at base of ram
[new_target_name] configure -work-area-virt 0 -work-area-phys 0x20000000 
-work-area-size 0x0800 -work-area-backup 0

Works fine.  Thanks!
