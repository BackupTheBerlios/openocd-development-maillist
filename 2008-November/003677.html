On Nov 19, 2008, at 4:13 PM, Steve Franks wrote:

> openocd -interface ft2232 -ft2232_layout "olimex-jtag" -reset_config  
> none -jtag_device 4 0x1 0xf 0xe -target create target0 arm7tdmi - 
> endian little -chain-position 0 -variant arm7tdmi-s_r4 -flash bank  
> lpc2000 0x0 0x7d000 0 0 0 lpc2000_v2 14745 -arm7_9 dcc_downloads  
> enable -flash probe 0 -flash auto_erase on -flash write_image $ 
> (TARGET).elf
>

99% of this can be done with a config file and the -c option.   
Something like:

openocd -f interface/olimex-jtag-tiny-a.cfg -f target/lpc2129.cfg -c  
'flash write_image $(TARGET).elf'

That would use the included olimex interface config and the lpc2129  
target config and simply run the flash write.  You could even add a  
quit to the end so the daemon terminates rather than waiting for  
connections.
