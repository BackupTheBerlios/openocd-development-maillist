Hi,

This is a config file to iMX27:

diff -Naur a/src/target/target/imx27.cfg b/src/target/target/imx27.cfg
--- a/src/target/target/imx27.cfg       1969-12-31 21:00:00.000000000 -0300
+++ b/src/target/target/imx27.cfg       2008-11-01 12:52:30.000000000 -0200
@@ -0,0 +1,12 @@
+#use combined on interfaces or targets that can't set TRST/SRST separately
+reset_config trst_and_srst
+
+# There are 2 taps on the chip:
+# The ETM
+jtag_device 4 0x1 0xf 0xe
+# The ARM926EJS
+jtag_device 4 0x1 0xf 0xe
+
+# Note above there are 2 taps (#0 and #1) the ARM926 is the 2nd tap (ie #1)
+target arm926ejs little 1 arm926ejs
+

This patch is based on previous config file posted by Duane.

We (me, Fabio and Rogerio) got iMX27ADS board working fine with this
config file.

I wrote a step-by-step tutorial and a simple ledtest application to
demonstrate it working:
http://www.imxdev.org/wiki/index.php?title=I.MX27_ADS_Board_JTAG_OpenOCD
