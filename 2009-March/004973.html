About which ID code do you talk here? About the jrc or cpu one? Why do

> Can we guarantee that the JRC

Why? Can we guarantee that all .cfg are different? If not, you will 
end with omap3530.cfg, omap3503.cfg, omap3515.cfg, omap3525.cfg, same

I vote for starting with omap3.cfg expecting that all configs are the 
same and change it if we find something different.
