>Error: JTAG tap: lpc2478.cpu             got: 0x4f1f0f0f (mfg: 0x787, part: 0xf1f0, ver: 0x4)

put that value (0x4f1f0f0f) in your config file at the top:
set CPUTAPID 0x4f1f0f0f

0xFFFFFFFF is the default value used to cause the error, so that you can get the right value.
