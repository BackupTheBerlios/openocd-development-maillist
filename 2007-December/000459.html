Hello Massimo,

as I understand it correct, the 28F640 should work without
modification of the OpenOCD. I have here two boards one
with TE28F160 and the other with a TE28F320. Both are
working.

I have used the following settings:

# TE28F160
flash bank cfi 0x80000000 0x200000 2 2 0

# TE28F320
flash bank cfi 0x80000000 0x400000 2 2 0

Do you know if the external memory interface is enabled
to get connection to the flash? With my LPC boards I must
enable the external memory interface before I can use the
flash.
