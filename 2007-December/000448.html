Hello List,

take a look in the cfi_flash_bank_command. Here at line 614
the cfi_infi will be allocated. The cfi_info had some function pointer.

working_area_t *write_algorithm;
working_area_t *erase_check_algorithm;

This pointer will not be set from the cfi_flash_bank_command function.
If I set erase_check_algorithm to 0, the "flash erase_check" will not crash.

I think we should set both, write_algorithm and erase_check_algorithm to 0.
