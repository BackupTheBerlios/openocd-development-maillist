Are you absolutely sure that your code was loaded in the chip? Because Eclipse takes the instructions from the chip and these seem totally different than what you'd expect... You can verify the instructions at some address with OpenOCD via telnet with command "arm disassemble <address> <count> thumb" - you will know then whether your code is flashed in the chip.
