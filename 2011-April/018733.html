I do not have an STM32 target with me. But I just tried to use
J-Link V3 and V7 under Windows 7 32bit and it works with
the Olimex LPC-P2148 target (using libusb-win32 filter
driver and the original Segger driver).

I've seen this before but I thought the issue has been solved
long before.

What is the Linux distro you are using (assuming that you
are using Linux). Are you using libusb-0.1 or libusb-1.0 plus
libusb-compat-0.1? Debian/Ubuntu seem to use libusb-0.1
and some other distros use libusb-1.0 plus libusb-compat-0.1.
