Try deleting c:\cygwin and reinstalling Cygwin.

You should budget (time) to wipe the entire cygwin directory
every so often. It helped me with a build problem today....

My laptop at home just segfaults willy nilly when using Cygwin
(even after wiping c:\cygwin), so my experience is that stars must
align before Cygwin works...

E.g. if you run setup.exe you get something different *every* day :-) as
there is no concept of releases in Cygwin, like you have for
e.g. Debian.
