On Saturday 13 June 2009, Xiaofan Chen wrote:
> I do not use FTDI based Jtag tools. But I think libftdi is based on libusb.
> So it will not work under Vista 64 bit (and the upcoming Windows 7 64bit)
> since libusb-win32 does not work under Vista 64.

Which means that anyone wanting OpenOCD on a 64-bit platform
must currently either (a) upgrade from Windows (to e.g. Linux,
BSD, etc) so they can use someone else's pre-built binaries
as part of a distribution; or else (b) build a private copy.


> FTDI's D2XX driver and DLL support Vista 64.

Great, but no distributor is allowed to provide a version
of OpenOCD which uses that.  Private builds only.
