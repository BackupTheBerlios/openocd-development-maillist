Hello List,

here is the inf file package for the jtagkey and turtelizer
for a 32-bit windows version only.

Before you can install a libusb version over an existing 
ftd2xx version you must uninstall the ftd2xx before. This
can be done with FTClean which can be found here:

http://www.ftdichip.com/Resources/Utilities.htm

Your FT2232 device must be unpluged before uninstall the old driver!
Now you can plug in your device and windows will need a new driver.
Point to the libusb driver and everything should work.

With my inf files I use the compoite version like:

[Devices]
"Amontec JTAGkey ( Channel A )"=LIBUSB_DEV, USB\VID_0403&PID_cff8&MI_00
"Amontec JTAGkey ( Channel B )"=LIBUSB_DEV, USB\VID_0403&PID_cff8&MI_01

The other one with only one device:

[Devices]
"Amontec JTAGkey"=LIBUSB_DEV, USB\VID_0403&PID_cff8

was not working here.

I think we should start to collect the inf files too?

Best regards,

Michael
