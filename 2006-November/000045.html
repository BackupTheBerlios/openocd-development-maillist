The target_read_uxx functions has no mechanism for checking the validity 
of returned data.

If the address is not word aligned or there is an nonfatal read error 
then the  target->type->read_memory function returns an ERROR value but 
this is dropped by target_read_uxx and not possible to check for from 
the calling function. In some of these cases a warning message is 
generated but not always.
