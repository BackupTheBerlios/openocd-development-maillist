On Thu, Aug 5, 2010 at 8:09 PM, simon qian <simonqian.openocd at gmail.com> wrote:
> I have also received requests for driver under Win7 64-bit.
> As the user says:Installing the Versaloon drivers via libusb on Windows 7 64
> seems to be VERY complex.
>
> I'll test it as soon as I get a Win7 64-bit.

Now it should be very simple with the new inf-wizard in release 1.2.1.0.
Actually the installation of the driver under 32bit or 64bit Windows are the
same now. You can try it under 32bit Windows first.

I would suggest you guys try to just distribute inf-wizard.exe with your
hardware and not the inf files.

Before 1.2.0.0, the libusb0.sys is not digital signed. So it is really
complex if you want to install under 64bit Vista/Win7 since you
need to mess with disable digital signing or with test signing.

Now you do not need all those tricks since the kernel driver libusb0.sys
is digital-signed with GlobalSign digital certificate.
http://www.globalsign.com/code-signing/
http://www.microsoft.com/whdc/driver/install/drvsign/kmcs-walkthrough.mspx
