Thanks. Take note the x86 version of inf-wizard.exe is based on
libwdi. The best thing about this current inf-wizard.exe is that
it has embedded x86 and 64bit ( x64 and ia64) version of driver
installer and the x86/x64/ia64 version of the driver/dll. So it actually
installs the correct driver/dll under x86, x64 and ia64.

The ia64 version is not tested since none of us have an Itanium
based computer ;-)

For x64/ia64, the 32bit dll is also installed (in syswow64 directory)
along with the 64bit driver/dll. This will enable 32bit OpenOCD
to run properly as well.
