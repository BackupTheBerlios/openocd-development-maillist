Hi,

I submitted this patch some time ago, but at the time the 0.4.0 release 
branching was about, so the it was decided to wait.

Now I would like to submit the patch again for better luck aiming at 0.5.0 
trunk.

The patch improves flash erase for STR9x in case of a full bank erase.
Then the chip erase command is used instead which improves speed 
significantly.

Also I think it might help if e.g. STR912 enters some state where flash 
banks are locked, and a chip erase command is the key for unlocking the 
flash.
