> I can run tests and report the result but there is simply no time for me

> and preferably one that don't mess up the system. That is, installs in
> some place that can easily be removed.
>
> I prefer to use Linux since libusb, libftdi and openocd is more native
> on that platform.

Under Linux, it is really not difficult to build OpenOCD. Probably
you want to try it.

libusb-1.0 should come with your Fedora 12.
You can remove the libftdi package from your Fedora 12. Then you
can install libftdi-1.0. By default it is installed in /usr/local. And
then you can build OpenOCD, by default it is also installed in
/usr/local. So they will not mess up your system.
