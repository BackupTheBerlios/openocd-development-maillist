Hello Carsten,

the i.MX31 (All I could find was the i.MX31, no i.MX32) isn't supported as it 
has an ARM1136JF-S core. The i.MX21 has an ARM926EJ-S core and is supported.

The Cirrus EP9315 has an ARM920t core and is supported. You should check the 
errata sheets for the various silicon revisions, versions prior to E2 have a 
bug that leads to lockups during start that needs to be worked around. 
There's also a problem with the RTC oscillator (iirc), but these problems are 
explained in the design guidelines.

The AT91SAM926x has an ARM926EJ-S core and is supported.

The support for ARM920t is quite mature and there shouldn't be any problems 
with that core.
The support for ARM926EJ-S (cache & mmu handling) is quite new and not fully 
tested (few users have a ARM926EJ-S based target), but it should work, and in 
my tests everything worked fine so far.

Dominic
