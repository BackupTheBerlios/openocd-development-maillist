Hello Vincent,

thank you a lot for this work, I've applied all of your warning fixes.

I usually configure my OpenOCD working tree with
CFLAGS="-O0 -Wall -g3" ./configure ...

I think that users don't want to see leftover warnings when they just want to 
compile and use the OpenOCD.
For a developer, manually specifying the CFLAGS also allows you to reduce 
optimization and increase debug level.

Regards,

Dominic

> It remains several noisy "zero-length printf format string" warning (which
