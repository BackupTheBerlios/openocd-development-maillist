Oleg Kravchenko wrote:
> I can't setup proper config file

It's a good idea to avoid copypasting from existing interface and/or
target config files. Just use source to pull them in to your config:

source [find interface/parport.cfg]
source [find target/pxa270.cfg]

It seems that parport.cfg does not fit your cable, so please create a
new file in interface/ for your interface and send a patch! :)
