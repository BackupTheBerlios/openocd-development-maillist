 @subsection ARM11 specific commands
 @cindex ARM11
 
+ at deffn Command {arm11 asid} <value>
+Enables debugging of specific user land application whose Application
+Space Identifier equals @var{value}. This is achieved by linking all
+hardware breakpoint (BPR's) against one context aware breakpoint (BRP).
+This in turn puts CPU into debug state only when CPU fetches instruction
+from particular address and the current context id equals @var{value}.
+
+Setting @var{value} to 0 disables context aware debugging.
+ at end deffn
+

 Displays the value of the memwrite burst-enable flag,
