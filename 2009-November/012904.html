Start with creating a git patch against 0.3.1. This is actually quite easy,
but will require that you google up the very basics of git.

1. Run git clone

2. Create your very own branch:

git checkout -b aducflash v0.3.1

3. Copy all the files from your modified 0.3.1

4. Create the git patch:

git format-patch v0.3.1 --author="Guenther Enrico <guenther at striges.de>"

5. post the patch to the list.

Next you need to run git rebase to update your patches to the latest openocd
version(git rebase origin/master), but post the patch #5 first and we'll knudge
you along.
