Patch enables user land debugging per individual process. This is
achieved by linking context aware/unaware BRPs with one context aware
BRP. The former is firing when cpu fetches instructions from
particular IMVAs, the latter is monitoring register 13 in cp15 on
ARM11 (this is the place where context id / asid is held). When two
conditions are satisfied core enters debug mode.
