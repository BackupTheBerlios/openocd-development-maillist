>
> Not today.  The parameter to "step" is an address to resume
> from, not a count.
>
> I've wanted such an option too though ... e.g. to go through
> the end of a loop.
>

proc fwd {n} {
   for {set i 0} {$i < n} {set i [expr $i+1]} {
     step
   }
}
